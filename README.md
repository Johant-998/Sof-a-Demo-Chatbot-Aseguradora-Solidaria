# 🛡️ Sofía — Demo Chatbot Aseguradora Solidaria

Demo interactivo de un asistente virtual de seguros, simulado sobre la interfaz real de [Aseguradora Solidaria de Colombia](https://aseguradorasolidaria.com.co/).

## 🚀 Ver demo
👉 [Abrir en GitHub Pages](https://tu-usuario.github.io/nombre-del-repo/)

---

## ✨ Características

- 💬 Chatbot flotante posicionado sobre captura real del sitio oficial
- 🧠 Base de conocimiento extraída directamente del sitio web real
- 🛡️ Cubre: hogar, SOAT, vehículos, mascotas, arrendamiento, accidentes personales, siniestros y pagos
- ⚡ Respuestas con quick replies contextuales
- 🔄 Indicador de escritura animado
- 🪶 100% vanilla: sin frameworks ni dependencias externas

---

## 📁 Estructura del proyecto

```
├── index.html            # Estructura principal del demo
├── styles.css            # Estilos del chatbot y escena

```

## ⚙️ Cómo usar localmente

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/nombre-del-repo.git
   ```
2. Entra a la carpeta:
   ```bash
   cd nombre-del-repo
   ```
3. Abre `index.html` directamente en tu navegador.

> No requiere servidor ni instalación de dependencias.

---

## 🚀 Despliegue en GitHub Pages

1. Ve a **Settings → Pages** en tu repositorio
2. En **Source**, selecciona la rama `main` y la carpeta `/ (root)`
3. Guarda y espera unos minutos
4. Tu demo estará disponible en: `https://tu-usuario.github.io/nombre-del-repo/`
